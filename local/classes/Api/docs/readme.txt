Все запросы обрабатываются по адресу /api/index.php?path={название_метода}
Параметры методов можно узнать в соответствующих .yaml файлах в этой папке

Методы:
detail - детальная информация для конкретного элемента
documentReference - свойства элемента, которые используются для "Справка документа"
search - поиск элементов по фильтру
feedbackForm - сохранение данных формы в инфоблок
getCaptcha - возвращает информацию о капче
getListValues - возвращает возможные значения свойства типа список по коду и инфоблоку
getRubricValues - возвращает значения рубрик
getCategoryList - возвращает возможные категории по инфоблокам
